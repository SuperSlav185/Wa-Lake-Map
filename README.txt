Washington Lake Contour Map
============================
This project is a starter interactive map showing lake contours for WA state.

FILES:
- index.html: Main webpage
- style.css: Map styling
- script.js: Leaflet map logic
- lakes.geojson: Sample lake data (Lake Chelan)
- README.txt: Instructions

TO DEPLOY:
1. Create a GitHub repo.
2. Upload these files.
3. Go to Settings > Pages > Set source to "main" branch.
4. Visit your public link!
